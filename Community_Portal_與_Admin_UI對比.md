## 長者管理對比

### ✅ 相同的設計

| 元素 | Admin | Community Portal |
|------|-------|------------------|
| 佈局 | Table 格式 | ✅ Table 格式 |
| 搜尋 | 頂部搜尋框 | ✅ 頂部搜尋框 |
| 新增/編輯 | Modal 彈窗 | ✅ Modal 彈窗 |
| 設備選擇 | 下拉選單 | ✅ 下拉選單 |
| 長者資訊 | 頭像+姓名+年齡 | ✅ 頭像+姓名+年齡 |
| 設備狀態 | 標籤顯示 | ✅ 標籤顯示 |
| 狀態標籤 | 彩色圓角標籤 | ✅ 彩色圓角標籤 |
| 最後活動 | 相對時間 | ✅ 相對時間 |
| 操作按鈕 | 編輯/刪除圖示 | ✅ 編輯/刪除圖示 |

### 差異（合理）

| 元素 | Admin | Community Portal | 原因 |
|------|-------|------------------|------|
| 社區欄位 | ✅ 有 | ❌ 無 | 只管理單一社區 |
| 批次選擇 | ✅ 有 | ❌ 無 | 資料量較少 |
