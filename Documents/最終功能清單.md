## LINE 通知（3種）

### 1. 今日首次活動通知（優先級1）

**標題**：`{長輩} 今日首次活動`  
**觸發**：長者設備今天第一次被偵測到  
**自動/手動**：自動  
**接收者**：社區成員

### 2. 通知點通知（優先級2）

**標題**：`新偵測通知`  
**內容**：`{長輩} 出現在 {位置} 附近`  
**觸發**：經過已勾選的 gateway  
**自動/手動**：手動設定（在 Community Portal 勾選）  
**接收者**：社區成員

### 3. 不活躍警報（定時）

**觸發**：定時任務（每天 00:00, 12:00, 18:00）  
**接收者**：Line OA 管理員

---

## Community Portal 頁面（6個）

1. **Dashboard**（首頁）- 數據總覽 + 報告匯出
2. **長者管理** - Table + Modal
3. **設備清單** - Table + 詳情 Modal
4. **通知記錄** - 歷史查詢
5. **通知點** - Table + 選擇 Modal
6. **長者詳情** - 基本資料 + 設備足跡

## 快速測試（5分鐘）

### 測試首次活動通知

1. 訪問：https://safe-net-tw.web.app/community
2. 確認長者有綁定設備
3. 使用 Admin Beacon Test 發送測試
4. 確認 LINE 收到「今日首次活動」通知
5. 同一天再發送 → 確認不會重複

### 測試 Dashboard

1. 前往 Dashboard
2. 查看所有指標
3. 點擊「匯出報告」
4. 下載並查看報告內容
